Project 1 ~ Web Site Critiques ~ Brent Clark

Questions

  1. What was your first impression of this site?
  2. How does this site organize the flow of information? (Do they have links on the left and text on the right? Some other scheme? No scheme?)
  3. Does the site structure make sense or does it confuse you? Why?
  4. Does the site graphics/layout/multimedia enhance the site or detract from it? Why?
  5. How long did it take for the home page to load? Was the load time reasonable?
  6. Was the text color and size readable? If a background color or image was used, did it enhance or detract from the site?
  7. Did the site have a navigation bar or other navigation tool? Was the navigation well done?
  8. Did the site have a color scheme or look and feel that was repeated on all the pages?
  9. How does the site respond to smaller window sizes (responsive design)?
  10. What does this site offer that was unusual or very well done?

Website 1 Critique

  1. My first impression of this site was that it was nice and presentable. It looked like it was easy to navigate on.
  2. This website has some links on the bottom, and left side of the home page. It also has some tabs at the top that you
    can click on that will take you to different pages. Within the other pages, there are more links that you can click on.
  3. The website structure makes sense to me because it is easy to know where everything is. I know that if I wanted
    to look at some recipes, I could click on the recipes tab at the top. Most everything is clearly labeled.
  4. I think that the graphics and layout on this website do not distract from the website itself. The colors and layout
    are very pleasing to the eye.
  5. It did not take the website long at all to load. It loaded very quickly. The website was loaded almost instantly.
  6. The text was very easy to read because it was a good font size and the color was not too dark, and not too light.
    The background was white so it did not distract from the website.
  7. This site has a navigation bar. It is easy to use and was designed well.
  8. The color scheme of this website is red, blue, and black.
  9. When put into a smaller window, the website still has most all the information on it. However, it does not have the
    navigation bar when in a smaller window.
  10. This site is for a grocery store, but it has a page that has recipes on it. I thougt this was unusual for a grocery store
    website because I haven't seen that on other store websites.

Website 2 Critique

  1. My first impression of this site was that it seemed slightly boring at first. I didn't quite know what it was. I only saw
    the words in the middle of the home page and didn't see the navigation bar.
  2. This site does not have any links anywhere on the page. The only links it has are the ones within the navigation bar.
  3. The structure of this website confuses me a little bit. It confuses me because I don't really know where things are and
    it is a little hard to understand the website in general.
  4. I think that the layout and pictures distract from the website because that is really the only thing that I see. Until you
    scroll down, you can't actually see the words, only the pictures.
  5. The load time was very fast. I didn't have to wait long for it to load.
  6. The text is not very readable on the top of each page. The text is white which makes it hard to read against the background.
    The background is a picture on every page and I feel like it detracts form the website. I feel like all that you see is the picture.
  7. This website has a navigation bar. It is hard to see because it blends in with the background and it looks like any other
    text on the page.
  8. The whole website is black, gray, white, and purple. It has a darker theme.
  9. This site responds very well to smaller window sizes. The navigation bar is still there. It is just condensed down to
    three little lines in the corner. All of the text still fits on the page and you can still see the picture pretty well.
  10. Something good that they did was make the background black as you scroll down so that you can actually read what
    the text says.

Website 3 Critique

  1. My first impression of this website was that it looked very nice and easy to navigate on.
  2. This website has links on the left side of the page. It also has some links in the middle of the page.
  3. The structure of this website makes sense to me because it is not very cluttered and it is very easy to look up
    information.
  4. The graphics and layout help the website because it is very pleasing to the eye. The graphics are simple but
    still helpful to the website.
  5. The load time was not very long for this website. It loaded pretty quickly.
  6. The text was very easy to read and the background on each page helped the website because it gave you an
    idea of what all these places look like. It helps to know what the website is about as well.
  7. This website has a navigation bar and it is very well done. It is easy to tell what all of the tabs lead to. It is
    very easy to read and understand.
  8. This site has black text with simple and easy to read fonts. There are a lot of pictures on this website.
    There are pictures on every page.
  9. This website doesn't respond super well to smaller windows. It is hard to see everything that is displayed when
    you click on the browse button because the text overlaps other text.
  10. I think that this website in general was very well done. It has a very nice look to it and the pictures were very
    pretty and did not distract from the text.
